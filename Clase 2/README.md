# Cuestionario de Datos Personales y Hobbies
Hicimos un formularlio de datos personales y acitvidades personales.

## ¿Qué hace el código?

Se lre pregunta al usuario desde la terminal
* Nombre
* Edad
* Deporte favorito
* Comida favorita
* Película favorita

Una vez que el usuario ponga todas las respuestas, se recopilará todo en una frase en linea
